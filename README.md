### Find_Mine

## 필요 개선 사항

- 이미 단추가 열려 숫자가 적힌 둘레로 그 숫자만큼의 깃발이 세워져 있는 경우, 마우스 양쪽 버튼을 눌러 나머지(깃발이 세워져 있지 않은) 단추를 한꺼번에 열 수 있게 

- MenuBar 요소들 기능 모조리 추가하기

- 지뢰와 버튼 개수 사용자가 선택 가능하게

- 게임 끝났을 때 다시 시작 가능하도록
